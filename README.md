# Sales-Analysis-Dashboard-Excel
Sales Analysis Dashboard using Excel Pivot Tables &amp; Macros
Project Overview
This project is an interactive Sales Analysis Dashboard built entirely in Microsoft Excel using Pivot Tables, Pivot Charts, and Macros. The dashboard provides a comprehensive view of sales performance and customer behavior through dynamic visualizations and filters.
Tools & Techniques Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Excel Macros (VBA)
- Slicers & Timelines
- Data Cleaning and Transformation
Key Features
- Total Orders, Total Revenue, and Average Customer Spend
- Average Order-to-Delivery Time analysis
- Revenue analysis by Occasion, Category, Month, and Hour
- Top Products by Revenue
- Top Cities by Number of Orders
- Interactive slicers for Occasion and Date filtering
Objective
To demonstrate how Microsoft Excel can be used as a powerful data analysis and reporting tool to create automated and insightful dashboards that support data-driven decision-making.
Key Takeaway
This project highlights strong skills in Excel analytics, dashboard design, and business insight generation using Pivot Tables and Macros without relying on external BI tools.
